15.（5分）

设 AAA 为 n×mn\times mn×m 矩阵，BBB 为 m×nm\times nm×n 矩阵，并且 n<mn<mn<m。设 AB=EAB=EAB=E，其中 EEE 表示 nnn 阶单位矩阵，求证：BBB 的列向量组线性无关。

› 答案 / 解析

一个矩阵的列向量组线性无关的充要条件是矩阵的秩等于其列数。矩阵 BBB 是一个 m×nm \times nm×n 矩阵，其列数为 nnn。因此，我们只需证明 BBB 的秩 rank(B)=n\text{rank}(B)=nrank(B)=n。

根据题设 AB=EAB=EAB=E，其中 EEE 是 nnn 阶单位矩阵。单位矩阵的秩等于其阶数，所以 rank(E)=n\text{rank}(E)=nrank(E)=n。因此，rank(AB)=n\text{rank}(AB) = nrank(AB)=n。

根据矩阵秩的重要性质，两个矩阵乘积的秩不大于任何一个因子矩阵的秩，即 rank(AB)≤rank(B)\text{rank}(AB) \le \text{rank}(B)rank(AB)≤rank(B)。因此，我们有 n≤rank(B)n \le \text{rank}(B)n≤rank(B)。

矩阵 BBB 是一个 m×nm \times nm×n 矩阵，其秩不能超过其行数或列数，即 rank(B)≤min⁡(m,n)\text{rank}(B) \le \min(m, n)rank(B)≤min(m,n)。因为题目给定 n<mn < mn<m，所以 rank(B)≤n\text{rank}(B) \le nrank(B)≤n。

结合不等式 n≤rank(B)n \le \text{rank}(B)n≤rank(B) 和 rank(B)≤n\text{rank}(B) \le nrank(B)≤n 可得 rank(B)=n\text{rank}(B) = nrank(B)=n。

由于矩阵 BBB 的秩等于其列数 nnn，所以 BBB 的列向量组线性无关。证毕。
