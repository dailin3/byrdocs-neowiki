  • a 不去 P1P_{1}P1​。
  • b 不去 P3P_{3}P3​ 和 P4P_{4}P4​。
  • c 不去 P2P_{2}P2​。
  • d 不去 P1P_{1}P1​。

求可能的方法数。

› 答案 / 解析

我们将此问题模型化为一个 n×nn\times nn×n 的棋盘。棋盘的行代表人物，列代表地方。一个“不允许”的分配方案，我们称之为禁区，在棋盘上用 ■ 标记。

问题的解，等价于在这个 n×nn\times nn×n 棋盘上放置 nnn 个“车”，要求任意两个车不能在同一行或同一列，并且没有任何一个车落在禁区（■）中。

首先计算禁区的棋盘多项式 R(x)=∑k=0nrkxkR(x) = \sum_{k=0}^{n} r_k x^kR(x)=∑k=0n​rk​xk。其中，系数 rkr_krk​ 是指在禁区棋盘上放置 kkk 个互不攻击（不同行不同列）的车的方法数。

计算可得 r0=1r_0 = 1r0​=1， r1=5r_1 = 5r1​=5， r2=8r_2 = 8r2​=8， r3=4r_3 = 4r3​=4， r4=0r_4 = 0r4​=0。 因此，禁区的棋盘多项式为 R(x)=1+5x+8x2+4x3R(x) = 1 + 5x + 8x^2 + 4x^3R(x)=1+5x+8x2+4x3。

利用容斥原理，满足条件的排列总数可以通过公式 ∑k=0n(−1)krk(n−k)!\sum_{k=0}^{n} (-1)^k r_k (n-k)!∑k=0n​(−1)krk​(n−k)! 计算，代入 n=4n=4n=4 和 rkr_krk​ 的值，得到可能的分配方法总数为 666 种。
