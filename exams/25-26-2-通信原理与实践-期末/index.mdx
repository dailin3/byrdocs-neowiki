## 二

以 $m(t) = 4\cos(2\pi t) + 2\cos(20\pi t)$ 为基带调制信号，对载波 $\cos(2\pi f_c t)$ 进行调制。得到 $s(t)=m(t)\cos(2\pi f_c t)+\hat{m}(t)\sin(2\pi f_c t)$，其中 $\hat{m}(t)$ 是 $m(t)$ 的希尔伯特变换，$f_c$ 充分大。$s(t)$ 通过 AWGN 信道传输，到达收端的信号为 $r(t)=s(t)+n_W (t)$ 。其中噪声 $n_W (t)$ 的双边功率谱密度为 $N_0/2$。

1. 求 $m(t)$ 的功率
2. 求 $s(t)$ 的复包络 $s_L (t)$ 的表达式，$s_L (t)$ 的功率及功率谱密度
3. 画出相干解调器框图
4. 求理想相干解调器的解调输出信噪比
